# User Management REST API

This project is a User Management REST API built using Spring Boot, Spring Data JPA, Hibernate, and MySQL. It follows a layered architecture consisting of Controller, Service, Repository, DTO, and Exception Handling layers to ensure clean code organization and maintainability.
The application provides RESTful APIs for creating, retrieving, updating, and deleting user records. It demonstrates industry-standard practices such as DTO-based data transfer, JPA/Hibernate integration, global exception handling, and database persistence using MySQL.

## Technologies Used

* Java
* Spring Boot
* Spring Data JPA
* Hibernate
* MySQL
* Maven
* Lombok
* Postman

## Features

* Create, Read, Update, and Delete (CRUD) operations
* RESTful API design
* DTO Pattern implementation
* Global Exception Handling
* MySQL database integration
* Layered Architecture (Controller-Service-Repository)
* Postman API testing
